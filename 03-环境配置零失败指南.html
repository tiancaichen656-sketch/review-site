Python 环境配置零失败指南

适用：Windows / macOS / Linux 三平台 · 含 17 个常见报错解决方案

一、安装 Python

Windows

1. 打开 python.org → Downloads → 下载最新版
2. 双击安装包 → 勾选 "Add Python to PATH"（最重要！）
3. 点击 Install Now → 等待完成
4. 验证：打开 CMD（Win+R 输入 "cmd"），输入 python --version
报错："python 不是内部或外部命令"
→ 没勾选 Add to PATH。重新安装，或手动把 Python 路径加到系统环境变量。

macOS

1. 打开终端（Command+空格 → 输入 "Terminal"）
2. 安装 Homebrew：/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
3. brew install python@3
4. 验证：python3 --version

Linux (Ubuntu/Debian)

sudo apt update && sudo apt install python3 python3-pip -y
验证：python3 --version

二、安装 IDE（VS Code 推荐）

1. 打开 code.visualstudio.com → 下载安装
2. 打开 VS Code → 左侧 Extensions（Ctrl+Shift+X）→ 搜索 "Python" → 安装 Microsoft 官方的 Python 扩展
3. 创建第一个文件：File → New File → 保存为 hello.py → 输入 print("Hello Python!")
4. 右键 → Run Python File in Terminal
报错："No Python interpreter selected"
→ VS Code 底部点 Python 版本号 → 选择你安装的 Python

三、安装常用第三方库

# 打开终端/CMD，逐条运行：
pip install requests        # HTTP 请求（爬虫必备）
pip install pandas          # 数据分析
pip install matplotlib      # 数据可视化
pip install openpyxl        # Excel 操作
pip install beautifulsoup4  # HTML 解析（爬虫）
pip install flask           # Web 框架
报错："pip 不是内部或外部命令"
→ python -m pip install [库名]
报错："ERROR: Could not install packages due to an OSError"
→ 加 --user：pip install --user [库名]
报错："Microsoft Visual C++ 14.0 is required"
→ 下载 VS Build Tools：链接

四、17 个常见报错速查

报错信息原因解决方法
SyntaxError: invalid syntax语法错误（漏括号、引号等）检查错误行的上一行
IndentationError缩进不一致统一用 4 个空格
NameError: name 'xxx' is not defined变量未定义检查拼写或是否先赋值
TypeError: can only concatenate str字符串和数字混用用 str() 转换
ModuleNotFoundError库未安装pip install [模块名]
FileNotFoundError文件路径不对用绝对路径或检查文件名
PermissionError文件被占用或无权限关闭文件或管理员运行
IndexError: list index out of range列表索引越界检查列表长度
KeyError字典键不存在用 .get() 代替
AttributeError对象没有该属性检查变量类型
ImportError导入路径错误检查模块路径
ValueError值类型不对检查输入数据类型
ZeroDivisionError除以零加判断除数不为零
ConnectionError网络连接失败检查网络/URL
TimeoutError请求超时加 timeout 参数
UnicodeDecodeError编码问题指定 encoding='utf-8'
