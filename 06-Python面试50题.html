Python 面试 50 题精讲

每题含：题目→思路→代码→易错点。面试官真正想听什么。

1. Python 中 list 和 tuple 的区别？

答：list 可变（可增删改），tuple 不可变（创建后不能修改）。tuple 可作字典键，list 不能。tuple 内存占用更小、迭代更快。

面试官想听：你提到了"可哈希"和"内存效率"，而不是只答"可变/不可变"。

2. 什么是列表推导式？写一个例子。

列表推导式是创建列表的简洁语法。
[x**2 for x in range(10) if x % 2 == 0] → 生成 0-9 中偶数的平方。

3. Python 中 is 和 == 的区别？

== 比较值是否相等，is 比较是否同一个对象（内存地址）。小整数 [-5, 256] 会被 Python 缓存，所以 a=256; b=256; a is b 为 True，但 a=257; b=257; a is b 为 False。

这是高频题。面试官会用小整数的边界 256/257 来测试你是否真的理解。

4. Python 的垃圾回收机制？

① 引用计数为主（引用为 0 立即回收）② 标记清除处理循环引用 ③ 分代回收（0代/1代/2代，新对象在0代，存活越久代数越高，回收频率越低）。

5. 装饰器是什么？写一个计时装饰器。

import time
def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f'{func.__name__} 运行了 {time.time()-start:.2f}s')
        return result
    return wrapper

@timer
def slow_func():
    time.sleep(1)

6. *args 和 **kwargs 是什么？

*args 接收任意数量的位置参数（打包成 tuple），**kwargs 接收任意数量的关键字参数（打包成 dict）。用于函数参数数量不确定时。

7. Python 中深拷贝和浅拷贝的区别？

浅拷贝（copy.copy()）只复制对象本身，嵌套对象还是引用。深拷贝（copy.deepcopy()）递归复制所有嵌套对象，完全独立。

8. 解释 Python 的 GIL（全局解释器锁）？

GIL 是 CPython 的机制，同一时刻只有一个线程执行 Python 字节码。多线程 CPU 密集型任务无法利用多核。解决方案：多进程（multiprocessing）或 IO 密集型用线程。

9. 如何反转一个字符串？

s[::-1] 或 ''.join(reversed(s))

10. 字典如何按值排序？

sorted(d.items(), key=lambda x: x[1], reverse=True)

11. Python 的 with 语句有什么用？

自动管理资源（文件、锁、连接等），确保使用后正确关闭。背后是 __enter__ 和 __exit__ 魔术方法（上下文管理器）。

12. 如何去除列表中的重复元素并保持顺序？

list(dict.fromkeys(mylist))（Python 3.7+ dict 保序）

13. lambda 函数是什么？

匿名函数，一行写完。常用于 sorted/map/filter 等需要小函数的场景。
sorted(students, key=lambda s: s['score'])

14. Python 的 pass 语句有什么用？

占位符。当语法需要语句但你不想执行任何操作时使用（如空函数体、空类、异常处理中不处理时）。

15. Python 中如何实现单例模式？

class Singleton:
    _instance = None
    def __new__(cls, *args, **kwargs):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
